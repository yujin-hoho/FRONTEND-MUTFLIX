# Personal Streaming App / Aplikasi Streaming Pribadi

## 🇮🇩 Tentang Proyek

Proyek ini dibuat dengan tujuan agar siapa pun dapat memiliki aplikasi streaming pribadi mereka sendiri.

Aplikasi ini tidak menyediakan konten apa pun yang melanggar hak cipta. Proyek ini juga tidak ditujukan untuk distribusi konten ilegal, pembajakan, atau penggunaan komersial tanpa izin.

Website ini secara default tidak dibuka secara online. Namun, kamu dapat mengambil source code dari proyek ini dan menjalankannya di mesin atau server pribadi milikmu sendiri.

## Konten

Mungkin kamu akan bertanya:

> Jika aplikasi streaming ini tidak menyediakan konten apa pun, apakah masih bisa disebut aplikasi streaming?

Jawabannya: bisa.

Kamu dapat mengisi aplikasi ini dengan konten yang kamu miliki sendiri, seperti video pribadi, film buatan sendiri, dokumentasi, arsip media, atau konten lain yang memang kamu punya hak untuk menggunakannya.

Saya tidak menyediakan, mengarahkan, ataupun bertanggung jawab atas sumber konten yang kamu gunakan. Segala risiko hukum, teknis, dan penggunaan konten sepenuhnya menjadi tanggung jawab masing-masing pengguna.

## Penyimpanan File Konten

Idealnya, file konten seperti video, subtitle, dan thumbnail diletakkan di CDN atau layanan penyimpanan yang memang dirancang untuk kebutuhan streaming.

Namun, karena proyek ini dirancang sebagai aplikasi streaming versi low-budget, bahkan hampir zero-budget, Google Drive dapat digunakan sebagai salah satu opsi penyimpanan alternatif.

Perlu diingat bahwa performa Google Drive tidak akan sebaik CDN profesional. Kecepatan akses, limit bandwidth, stabilitas, dan pengalaman streaming dapat berbeda-beda tergantung kondisi penggunaan.

## Tujuan Proyek

Proyek ini tidak dimaksudkan untuk menggantikan layanan streaming profesional seperti Netflix, Disney+, Prime Video, dan sejenisnya.

Tujuan utamanya adalah memberikan pengalaman membangun aplikasi streaming pribadi dengan biaya seminimal mungkin, sehingga pengguna dapat memiliki semacam “Netflix pribadi” untuk konten yang mereka miliki sendiri.

## Disclaimer

Proyek ini hanya menyediakan source code aplikasi.

Saya tidak menyediakan konten video, film, serial, musik, atau media lain yang dilindungi hak cipta.

Dengan menggunakan proyek ini, kamu setuju bahwa seluruh konten yang kamu tambahkan, tampilkan, atau bagikan melalui aplikasi ini merupakan tanggung jawabmu sendiri.

Gunakan proyek ini dengan bijak dan patuhi hukum yang berlaku di wilayah masing-masing.

# Personal Streaming App

## 🇬🇧 About This Project

This project was created with the goal of allowing anyone to have their own personal streaming application.

This application does not provide any copyrighted or illegal content. It is not intended for piracy, illegal content distribution, or unauthorized commercial use.

By default, this website is not deployed publicly online. However, you are free to clone the source code and run it on your own local machine, private server, or hosting environment.

## Content

You might ask:

> If this streaming application does not provide any content, can it still be called a streaming app?

The answer is: yes.

You can add your own content to the application, such as personal videos, self-produced films, documentation, media archives, or any other content that you legally own or have permission to use.

I do not provide, recommend, or take responsibility for the source of any content used with this project. Any legal, technical, or usage-related risks are entirely the responsibility of each user.

## Content File Storage

Ideally, media files such as videos, subtitles, and thumbnails should be stored on a CDN or a storage service designed for streaming purposes.

However, since this project is designed as a low-budget — or almost zero-budget — streaming application, Google Drive can be used as an alternative storage option.

Please note that Google Drive will not provide the same performance as a professional CDN. Streaming speed, bandwidth limits, stability, and overall user experience may vary depending on your usage and environment.

## Project Goal

This project is not intended to replace professional streaming services such as Netflix, Disney+, Prime Video, or similar platforms.

The main goal is to provide a simple and affordable way to build a personal streaming application, giving users a “personal Netflix-like” experience using their own content.
